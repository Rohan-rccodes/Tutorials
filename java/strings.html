Strings in Java

Strings in Java are sequences of characters used to represent text. Strings is a inbuilt class in java present in package java.lang.

When we create a object of an String, that object is immutable which means whenever we assign a value in the string object, that value is not overwritted in the same object instead new memory is allovated with a new assignment of a value.

String Memory and Accessibility

In Java, strings are stored in the string pool, a special area of memory. Strings are immutable, which means their values cannot be changed after creation. This immutability ensures that strings are thread-safe.

String objects can be created in several ways:

  • String Literal: String literals, such as Hello, Java!, are stored in the string constant pool and are automatically reused if the same literal is encountered. And the object reference of that will refer to the memory in the string constant pool.
  • Using the new Keyword: The new keyword creates a new string object in the heap memory, even if an identical string already exists in the string pool. And the object reference of string refers to the memory present in the heap.

String Creation and Accessibility

Here's an example of creating and accessing strings in Java: 

String literal1 = "Hello, Java!"; // Stored in the string pool
String literal2 = "Hello, Java!"; // Reuses the existing string in the pool
String newString = new String("Hello, Java!"); // Creates a new string object in heap memory

System.out.println(literal1 == literal2); // true (same reference)
System.out.println(literal1 == newString); // false (different references)

The StringBuffer Class

The StringBuffer class is a mutable sequence of characters that allows you to modify the content of a string without creating new string objects. It is more efficient for extensive string manipulation. By default, capacity of StringBuffer object is 16 characters. If the number of characters exceeds the capacity, the capacity is automatically increased at runtime.

Example of Using StringBuffer

Here's an example of using the StringBuffer class to concatenate strings: 

public class StringBufferExample {
    public static void main(String[] args) {
        StringBuffer buffer = new StringBuffer();
        buffer.append("Hello, ");
        buffer.append("Java!");
        String result = buffer.toString(); // Convert StringBuffer to String
        System.out.println(result); // Output: Hello, Java!
    }
}

Conclusion
