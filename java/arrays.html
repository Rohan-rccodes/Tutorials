Arrays

Arrays are homogeneous data structures that store a fixed-size, ordered collection of elements of the same type. They are widely used to store and manipulate data efficiently. Arrays in java are dynamic and continous. While declaring the array in java, we can't give the size of the array.

Types of Arrays

There are two main types of arrays in Java:

  • Single-Dimensional Arrays: These are arrays that have a single dimension or a single row. They are used to store a list of values of the same type.
  • Multi-Dimensional Arrays: These are arrays with two or more dimensions, often referred to as matrices. Multi-dimensional arrays can be used to represent tables, grids, and more complex data structures.

Steps for Working with Arrays

To work with arrays in Java, you typically follow these steps:

  1. Declaration: Declare an array variable with its type and size.

    Syntax :

    datatype array_name[]; //for single dimension
datatype array_name[][]...; //for multi dimension
  2. Memory Allocation: Allocate memory for the array using the new keyword.

    Syntax :

    array_name = new datatype[size]; //for single dimension
array_name... = new datatype[size1][size2]...; //for multi dimension
or
array_name = new datatype[size1];
array_name[0] = new datatype[size2];
  3. Assigning Values: Assign values to array elements using index notation.

    Syntax :

    array_name[index] = value;
  4. Accessing Elements: Retrieve and manipulate elements by their index.

    Syntax :

    array_name[index];

Example of Single-Dimensional Array

Here's an example of a single-dimensional array in Java: 

int[] numbers = new int[5]; // Declaration and memory allocation
numbers[0] = 1; // Assigning values
numbers[1] = 2;
numbers[2] = 3;
numbers[3] = 4;
numbers[4] = 5;
int thirdElement = numbers[2]; // Accessing elements
System.out.println("Third Element: " + thirdElement); // Output: Third Element: 3

Example of Jagged Array

A jagged array is an array of arrays, where each element can be an array of different sizes. Here's an example: 

int[][] jaggedArray = new int[3][]; // Declaration and memory allocation
jaggedArray[0] = new int[3];
jaggedArray[1] = new int[4];
jaggedArray[2] = new int[2];

Conclusion

Arrays are fundamental data structures in Java, allowing you to efficiently store and manipulate collections of data. Understanding the steps for declaration, memory allocation, assigning values, and accessing elements is essential for working with arrays in Java.
