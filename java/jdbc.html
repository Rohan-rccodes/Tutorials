Java DataBase Connectivity in Java

Java Database Connectivity (JDBC) is a mechanism in which we connect the Java API with the database.

For connecting the Java API with database, we first need the database driver. Databse driver is provided by company of database.

Steps for Query Execution

Executing a query in JDBC involves several steps: creating a statement, executing the query, and handling the results.

  • Step 1 - Load JDBC Driver: Load the database-specific JDBC driver using Class.forName(). For example we use mysql as database 
    Class.forName("com.mysql.jdbc.Driver");
  • Step 2 - Establish Connection: Create a connection to the database using the driver manager. 
    Connection con = DriverManager.connection(url, username, password);
  • Step 3 - Create a Statement: Create a statement object to execute SQL queries. 
    Statement st = con.createStatement();
  • Step 4 - Execute Query: Execute the SQL query using the statement object. For example if we execute insert query. 
    int affectedRow = st.executeUpdate(query);
  • Step 5 - Process Results: Retrieve and process the results obtained from the query. 
    ResultSet rs = st.executeQuery(query);
rs.next();
int id = rs.getInt("id");
  • Step 6 - Close Resources: Close the result set, statement, and connection to release resources. 
    con.close();

Example of Establishing a Database Connection

To connect to a database using JDBC, you need to establish a connection. This involves specifying the database URL, username, and password, and then creating a connection object. 

String url = "jdbc:mysql://localhost:3306/mydb";
String username = "user";
String password = "password";
Connection connection = DriverManager.getConnection(url, username, password);

Example of Executing SQL Queries

After establishing a connection, you can execute SQL queries on the database. This involves creating a statement, executing the query, and retrieving the results. 

Statement statement = connection.createStatement();
ResultSet resultSet = statement.executeQuery("SELECT * FROM users");
while (resultSet.next()) {
    // Process the result data
}
resultSet.close();
statement.close();

Conclusion

JDBC is a fundamental technology for database interaction in Java applications. It allows developers to connect to databases, execute queries, and retrieve data seamlessly.
