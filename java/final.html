Final Keyword

In Java, the "final" keyword is used to indicate that an element (variable, method, or class) is unchangeable or cannot be overridden or extended. Once marked as "final," the element's value or behavior cannot be modified or extended by subclasses.

Using final with Variables

When applied to a variable, the "final" keyword makes it a constant or read-only variable. Once assigned a value, a final variable cannot be reassigned. 

class Constants {
    final int MY_CONSTANT = 42;
}

Using final with Methods

When applied to a method, the "final" keyword prevents the method from being overridden by subclasses. It ensures that the method's behavior remains constant. 

class Base {
    final void myMethod() {
        System.out.println("This method cannot be overridden.");
    }
}
class Derived extends Base {
    // Error: Cannot override the final method myMethod()
    void myMethod() {
        System.out.println("This won't compile.");
    }
}

Usingfinal with Classes

When applied to a class, the "final" keyword indicates that the class cannot be extended or subclassed. It is used to prevent further inheritance. 

final class FinalClass {
    // Class content
}
// Error: Cannot inherit from final class FinalClass
class SubClass extends FinalClass {
    // This won't compile.
}

Using final with Constructors

While you cannot declare a constructor as "final" directly, making a class "final" indirectly makes its constructors "final". This means that you cannot extend a final class, and therefore, you cannot create new subclasses with different constructors. 

final class ImmutableClass {
    private final int value;
    public ImmutableClass(int value) {
        this.value = value;
    }
    // Getters
}
// No subclasses with different constructors can be created.

Conclusion

The "final" keyword in Java is a powerful tool for ensuring immutability, constant values, and preventing modification or extension of elements like variables, methods, and classes. It helps in creating more robust and secure Java programs.
