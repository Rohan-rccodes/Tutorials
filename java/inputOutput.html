Input and Output in Java

Input and Output (I/O) operations are essential for interacting with external data sources, such as files, the console, or network connections. Java provides various classes and methods for performing I/O operations.

In java, I/O classes are divided into two categories :

  1. Byte Oriented Classes : In these classes, the data would be read or write in the form of blocks or bytes (any size).

    The topmost B.O classes are InputStream and OutputStream. Both classes are abstract classes and present in java.io package.

  2. Character Oriented Classes : In these classes, the data would be read or write in the form of characters or strings.

    The topmost B.O classes are Reader and Writer. Both classes are abstract classes and present in java.io package.

We cannot take input from keyboard without using a object named as System.in.

Reading Data from the Console

You can read data from the console using the System.in stream and InputStreamReader class for character input. Here's an example of reading a line of text from the console: 

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class ConsoleInputExample {
    public static void main(String[] args) throws IOException {
        BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
        System.out.print("Enter a line of text: ");
        String input = reader.readLine();
        System.out.println("You entered: " + input);
        reader.close();
    }
}

Writing Data to the Console

You can write data to the console using the System.out stream and PrintWriter class for character output. Here's an example of printing a message to the console: 

import java.io.PrintWriter;

public class ConsoleOutputExample {
    public static void main(String[] args) {
        PrintWriter writer = new PrintWriter(System.out);
        writer.println("Hello, Java I/O!");
        writer.flush(); // Flush the output
        writer.close();
    }
}

Working with Files

Java provides classes like File, FileInputStream, FileOutputStream, BufferedReader, and BufferedWriter for working with files. Here's an example of reading and writing data to a text file: 

public class FileInputOutputExample {
    public static void main(String[] args) throws java.io.IOException {
        // Writing to a file
        java.io.BufferedWriter writer = new java.io.BufferedWriter(new java.io.FileWriter("example.txt"));
        writer.write("This is a sample text.");
        writer.close();

        // Reading from a file
        java.io.BufferedReader reader = new java.io.BufferedReader(new java.io.FileReader("example.txt"));
        String line;
        while ((line = reader.readLine()) != null) {
            System.out.println(line);
        }
        reader.close();
    }
}

Conclusion

Input and Output operations are fundamental for interacting with data in Java applications. Whether you need to read from the console or work with files, Java provides classes and methods to handle various I/O tasks efficiently.
