Constructors

Constructors in Java are special methods used for initializing objects. They are called when an object of a class is created. Constructors are used to set the initial state of an object, allocate memory, and perform other setup tasks.

Constructors are having some features :

  • They have same name as class name.
  • They does not have any return type.
  • We can pass arguments in the constructor.
  • If the programmer does not create any type of constructor then the compiler will create a public default constructor.

Types of Constructors

There are 3 types of constructors in Java:

  • Default Constructor
  • Parameterized Constructor
  • Copy Constructor

Default Constructor

If a class does not explicitly define any constructors, a default constructor is provided by the Java compiler. The default constructor takes no arguments and initializes instance variables to default values (e.g., 0 for numeric types, null for objects). 

class MyClass {
    int myVar;
    // Default constructor
    MyClass() {
        myVar = 42;
    }
}

Parameterized Constructors

Java allows you to define constructors with parameters, known as parameterized constructors. These constructors accept arguments and initialize instance variables based on the provided values. 

class Person {
    String name;
    int age;
    // Parameterized constructor
    Person(String n, int a) {
        name = n;
        age = a;
    }
}

Copy Constructor

A copy constructor is a constructor that takes an object of the same class as a parameter and creates a new object with the same values as the parameter object.

class Person {
    String name;
    int age;
    // Parameterized constructor
    Person(String n, int a) {
        name = n;
        age = a;
    }
    // Copy constructor
    Person(Person obj) {
        name = obj.n;
        age = obj.a;
    }
}

Using Constructors

To create an object and call a constructor, use the new keyword followed by the class name and constructor arguments (if any). Here's an example: 

public class Main {
    public static void main(String[] args) {
        // Using the default constructor
        MyClass obj1 = new MyClass();
        // Using the parameterized constructor
        Person person1 = new Person("Alice", 30);
    }
}

Constructor Overloading

Just like methods, constructors can also be overloaded. This means you can define multiple constructors with different parameter lists in the same class. 

class Book {
    String title;
    String author;
    // Parameterized constructor 1
    Book(String t, String a) {
        title = t;
        author = a;
    }
    // Parameterized constructor 2
    Book(String t) {
        title = t;
        author = "Unknown";
    }
}

Conclusion

Constructors in Java play a crucial role in object initialization and setting the initial state of objects. They can be used to provide default values or accept arguments to customize object initialization. Constructor overloading allows you to create objects with different initialization options.
