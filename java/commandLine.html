Command Line Arguments in Java

Introduction to Command Line Arguments

In Java, command line arguments are values provided to a Java program when it is invoked from the command line or terminal. These arguments allow you to pass information to your Java program before it starts execution. Command line arguments are commonly used for configuration, customization, or input data for Java applications.

How to Pass Command Line Arguments

To pass command line arguments to a Java program, follow these steps:

  1. Open a command prompt or terminal window.
  2. Navigate to the directory where your Java program (.class file) is located.
  3. Use the following syntax to run the Java program with arguments:
    4. $ java YourClassName arg1 arg2 arg3 ...
  4. Replace YourClassName with the name of your Java class that contains the main method.
  5. You can pass one or more arguments separated by spaces after the class name.

Accessing Command Line Arguments in Java

Inside your Java program, you can access command line arguments through the args parameter of the main method. The args parameter is an array of strings containing the command line arguments. 

public class MyProgram {
    public static void main(String[] args) {
        for (String arg : args) {
            System.out.println("Argument: " + arg);
        }
    }
}

Example Usage

Let's say you have a Java program named MyProgram, and you want to pass two arguments, arg1 and arg2. You would run it as follows: 

$ java MyProgram arg1 arg2 ...

In this example, the program would access and print the values of arg1 and arg2.

Conclusion

Command line arguments provide a way to customize and configure Java programs by passing input data from the command line. Java programs can access and process these arguments through the main method's args parameter.
