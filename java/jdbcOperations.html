JDBC Operations in Java

INSERT Operation

The INSERT operation is used to add new records to a database. Here's an example of inserting a new user into a database. 

import java.sql.Connection;
import java.sql.PreparedStatement;

String insertQuery = "INSERT INTO users (username, password) VALUES (?, ?)";
PreparedStatement preparedStatement = connection.prepareStatement(insertQuery);
preparedStatement.setString(1, "newuser");
preparedStatement.setString(2, "newpassword");
int rowsInserted = preparedStatement.executeUpdate();
preparedStatement.close();

SELECT Operation

The SELECT operation is used to retrieve data from a database. Here's an example of selecting users from a database. 

import java.sql.Statement;
import java.sql.ResultSet;

Statement statement = connection.createStatement();
String selectQuery = "SELECT * FROM users";
ResultSet resultSet = statement.executeQuery(selectQuery);
while (resultSet.next()) {
    // Process results
}
resultSet.close();
statement.close();

UPDATE Operation

The UPDATE operation is used to modify existing records in a database. Here's an example of updating a user's password. 

String updateQuery = "UPDATE users SET password = ? WHERE username = ?";
PreparedStatement updateStatement = connection.prepareStatement(updateQuery);
updateStatement.setString(1, "newpassword");
updateStatement.setString(2, "newuser");
int rowsUpdated = updateStatement.executeUpdate();
updateStatement.close();

DELETE Operation

The DELETE operation is used to remove records from a database. Here's an example of deleting a user from a database. 

String deleteQuery = "DELETE FROM users WHERE username = ?";
PreparedStatement deleteStatement = connection.prepareStatement(deleteQuery);
deleteStatement.setString(1, "newuser");
int rowsDeleted = deleteStatement.executeUpdate();
deleteStatement.close();

Conclusion

These JDBC operations in Java cover basic database operations: INSERT, SELECT, UPDATE, DELETE, and sorting. Understanding and mastering these operations is essential for working with databases in Java applications.
