Object-Oriented Programming (OOP)

Object-Oriented Programming (OOP) is a programming paradigm that uses objects and classes to structure code. In Java, OOP is based on four key principles:

  • Encapsulation: Wrapping data (attributes) and methods (functions) into a single unit called an object. Access to data is controlled through access modifiers.
  • Inheritance: Creating new classes (subclasses or derived classes) based on existing classes (superclasses or base classes) to inherit their properties and behaviors.
  • Polymorphism: The ability of objects of different classes to respond to the same method call in a way that is specific to their class. This is achieved through method overriding and method overloading.
  • Abstraction: Defining the essential characteristics of an object while hiding the unnecessary details. Abstraction is achieved through abstract classes and interfaces.

The Static Keyword in Java

In Java, the static keyword is used to create class-level members that belong to the class itself, rather than to instances (objects) of the class. Static members are shared among all instances of the class. There are several uses of the static keyword in Java, including:

  • Static Variables (Class Variables): Variables declared with the static keyword are shared by all instances of the class.
  • Static Methods: Methods declared as static belong to the class and can be called using the class name, without creating an instance of the class.
  • Static Blocks: Static blocks are used to initialize static variables or perform one-time class-level initialization.

Creating Objects in Java

  • In java, every object is anonymous.
  • Objects must be declared in head or dynamically.
  • Reference object get the reference of actual address, it cannot directly access actual memory

Here's an example of creating an object and understanding memory allocation: 

class Car {
    String brand;
    String model;
    void startEngine() {
        System.out.println("Engine started.");
    }
}
public class Main {
    public static void main(String[] args) {
        // Creating an object of the Car class
        Car myCar = new Car();
        // Memory allocation for the object myCar
        // Attributes: brand (null), model (null)
        // Method: startEngine()
    }
}

Memory which is not accessed by anyone or cannot be accessed because of lost of reference is known as Dereference object memory and in this situation, Garbage collector collects all the dereference memory and deallocat its memory.

Using Static Members

Here's an example of using static members in Java: 

class Counter {
    static int count = 0; // Static variable
    static void increment() { // Static method
        count++;
    }
}
public class Main {
    public static void main(String[] args) {
        Counter.increment();
        System.out.println("Count: " + Counter.count);
    }
}

Conclusion

Object-Oriented Programming (OOP) is a fundamental concept in Java that promotes code organization and reusability through classes and objects. The static keyword in Java allows you to create class-level members that are shared among all instances of a class, providing utility and efficiency in various programming scenarios.
