Packages

In Java, a package is a collection of multiple classes, interfaces and packages. Packages provide a mechanism to group classes that belong to the same domain or purpose, making code organization and management more efficient.

By default a package is imported in every source file know as java.lang.

Methods to represent classname

  • Qualified name : It is the absolute path of a class. For e.g. java.lang.String.
  • Unqualified name : In this method, first we import the package and then use it directly by name. For e.g. String, System.

Types of packages

  • Inbuilt package : All the inbuilt packages are written in lowercase letters. We can use or inherit any class of the inbuilt package depending upon its definition. To access a class of a package, we can either use a qualified name or an unqualified name by using import keyword.

    Example of qualified path name: 

    class A{
    public static void main(){
        java.lang.String s = "Hello";
        System.out.println(s);
    }
}

    Example of unqualified path name: 

    class A{
    public static void main(){
        String s = "Hello";
        System.out.println(s);
    }
}
  • User defined package: User defined packages can be named as uppercase and camelcase letters also. We can use them as same as Inbuilt packages.

Creating a Package

To create a package in Java, you include the package statement at the beginning of your source file. Packages are organized hierarchically, and you can use subpackages to further structure your code. 

package p1;
public class MyClass {
    // Class content
}

To compile the source file and move the bytecodes in the created folder of package name, we can use this command :

javac -d . file_name

Package inside a package

We can create a package inside a package like this

package p1.p2;

Using Packages

To use classes from a package, you either fully qualify the class name with the package name or use the import statement to bring specific classes or the entire package into your code. 

// Fully qualified class name
com.example.myapp.MyClass myObj = new com.example.myapp.MyClass();

// Using import statement
import com.example.myapp.MyClass;
// ...
MyClass myObj = new MyClass();

Java Standard Packages

Java provides a rich set of standard packages that cover various aspects of programming, such as java.util for utility classes, java.io for input and output, and java.lang for fundamental classes and exceptions. 

import java.util.ArrayList;
import java.io.FileReader;
// ...
ArrayList<String> list = new ArrayList<>();
FileReader fileReader = new FileReader("example.txt");

Conclusion

Packages in Java provide a structured way to organize classes and interfaces, making it easier to manage and maintain code. They are essential for building modular and maintainable Java applications and are widely used in both standard and custom libraries.
