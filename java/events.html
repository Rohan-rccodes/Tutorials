Event Handling in Java

Event handling is the mechanism through which we can handle the user acitvity perform on an object. Event is known as ac which is done by the user on a particular object. To handle the events, the language give us various evnet handlers/listeners. Event listeners are present in the form of interfaces in java. All the event listeners are present in a package java.awt.event. Every event listener is having one or more htna one function which the programmer can override depending upon the action.

Steps to implement Event Listener

  1. Implements the listener.
  2. Override all the functions.
  3. Register object to listener.
  4. Write action performed on that particular event.

Common Event Listeners

Action Listener

The ActionListener interface is used to handle actions, such as button clicks or menu selections. 

button.addActionListener(new ActionListener() {
    public void actionPerformed(ActionEvent e) {
        // Action handling code here
    }
});

Mouse Listener

The MouseListener interface is used to handle mouse-related events like clicks and movement. 

component.addMouseListener(new MouseAdapter() {
    public void mouseClicked(MouseEvent e) {
        // Mouse click handling code here
    }
});

Key Listener

The KeyListener interface is used to handle keyboard events, including key presses and key releases. 

component.addKeyListener(new KeyAdapter() {
    public void keyTyped(KeyEvent e) {
        // Key press handling code here
    }
});

Window Listener

The WindowListener interface is used to handle window-related events, such as window closing or activation. 

frame.addWindowListener(new WindowAdapter() {
    public void windowClosing(WindowEvent e) {
        // Window closing handling code here
    }
});

Adapter Classes

Adapter classes in Java provide default implementations for listener interfaces, making it easier to work with specific listeners. Common adapter classes include ActionAdapter, MouseAdapter, KeyAdapter, and WindowAdapter. 

button.addActionListener(new ActionAdapter() {
    public void actionPerformed(ActionEvent e) {
        // Action handling code here
    }
});

Conclusion

Event handling is a fundamental aspect of Java GUI development. By using event listeners and adapter classes, developers can efficiently respond to user actions and system events in Java applications.
