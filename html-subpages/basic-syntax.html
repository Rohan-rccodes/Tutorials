Basic Syntax

HTML documents have a basic structure...

Here is an example of a simple HTML document:

            
                <!DOCTYPE html>
                <html lang="en">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>My First HTML Page</title>
                </head>
                <body>
                    <h1>Hello, World!</h1>
                    <p>This is a paragraph.</p>
                </body>
                </html>
